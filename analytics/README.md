   - `participant_meta`

   - `raw_events.jsonl`

   - `adaptation_table.csv`
   - `session_table.csv`
   - `participant_mode_table.csv`

4. Открыть аналитические ноутбуки:
   - `analytics/05_1_compare_modes.ipynb`
